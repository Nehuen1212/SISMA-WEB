¿Qué es S.I.S.M.A.?

Una plataforma compacta para simular movimientos sísmicos y evaluar el comportamiento de maquetas bajo carga dinámica.

  • Simulación precisa basada en modelos empíricos
  • Control total de parámetros (intensidad, frecuencia, duración)
  • Registro y exportación de datos vía Wi‑Fi
  • Interfaz local y acceso remoto

Características

Simulación controlada

Parámetros finos para replicar distintos escenarios sísmicos.

Control remoto

Wi‑Fi integrado para monitoreo y control desde web o app.

Exporta resultados para análisis y comparativas.

Desarrollo técnico

Una plataforma de simulación sismica diseñada para evaluar la estabilidad estructural con control preciso y monitoreo en tiempo real.

Base fija

  • Estructura en aluminio 4040 de alta resistencia
  • Sistema de nivelación automática en las patas
  • Diseño modular para fácil mantenimiento
  • Acabado anodizado para mayor durabilidad

Sistema de movimiento

  • Motor NEMA 17 de alto torque (1.8° por paso)
  • Sistema de poleas GT2 para transmisión precisa
  • Plataforma móvil sobre guías lineales
  • Sensores de límite para seguridad

Control electrónico

  • ESP32-WROOM-32 (dual-core, WiFi+BLE)
  • Driver A4988 para control preciso del motor
  • Display OLED para configuración local
  • Sistema de alimentación redundante

Software y conectividad

  • API REST para control remoto
  • Interfaz web responsive
  • Almacenamiento de datos en tiempo real
  • Exportación en formatos CSV/JSON

Especificaciones técnicas

Rango de frecuencia 0.1 - 10 Hz
Amplitud máxima ±50 mm
Carga máxima 5 kg
Precisión 0.1 mm
Alimentación 12V DC / 5A
Conectividad WiFi 2.4GHz
