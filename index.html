Simulador Interactivo de Sismos

Para Maquetas y Analisis

S.I.S.M.A. redefine la prueba de estructuras. Diseñado para arquitectos y estudiantes, nuestro simulador ofrece precisión y control en cada simulación. Pensado para aquellos que quieren llevar la prueba de la estabilidad de su maqueta a otro nivel.

Descubre S.I.S.M.A.

¿Qué es S.I.S.M.A.?

S.I.S.M.A. es una herramienta innovadora para simular terremotos y evaluar la resistencia de maquetas de estructuras. Proporciona un entorno controlado para el análisis y la optimización de diseños.

Objetivos Clave

  • Simulación precisa de terremotos
  • Pruebas exhaustivas de maquetas
  • Control total de los parámetros sísmicos
  • Registro detallado de datos

Características Destacadas

Simulación Controlada

Ajusta la intensidad, frecuencia y duración de los terremotos con precisión milimétrica a travez de parametros que pueden ser controlados por el usuario en su totalidad.

Control Intuitivo

Interfaz amigable para una configuración rápida y sencilla de las simulaciones. Se puede configurar a travez del propio display manual que viene incorporado en el dispositivo o a travez de nuestra app mediante Wifi.

Registro de Datos

Almacena resultados de cada simulacion en una tabla que se puede observar desde una web o aplicacion, puediendo guardarlos y analizarlos o compararlos para optimizar tus diseños.

Desarrollo del Proyecto

Base Fija

Estructura y piezas robustas que proporcionan estabilidad al sistema, para una mayor seguridad y duracion a la hora de las simulaciones

Base Móvil

Plataforma dinámica impulsada por un sistema de poleas y motor Nema 17. Sobre esta se situa un canasto donde se incluira tierra u otro material sobre el que se desee construir la maqueta.

Microcontrolador ESP32

Gracias a este poderoso microcontrolador programable podemos obtener un control preciso de las simulaciones y gestión de datos.

Control Wifi

Operación remota y monitoreo a través de una aplicación móvil y una web, junto con una planilla, todo conectado vía wifi con el dispositivo para que este registre todas las simulaciones.

Simulación de Terremotos

Modelo matemático basado en la escala de Richter para simulación realista.
